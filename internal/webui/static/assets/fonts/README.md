Phragma WebUI font assets
=========================

These fonts are vendored from Google Fonts so the appliance UI does not depend
on external font CDNs at runtime:

- Chakra Petch, weights 500, 600, 700
- IBM Plex Sans, weights 400, 500, 600, 700
- IBM Plex Mono, weights 400, 500, 600

Chakra Petch, IBM Plex Sans, and IBM Plex Mono are distributed under the SIL
Open Font License. Keep these files local unless a future packaging plan adds a
different self-hosted font pipeline.
