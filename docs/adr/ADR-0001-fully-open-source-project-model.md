# ADR-0001: Fully Open Source Project Model

Status: Accepted

## Context

The project originally considered an open-core model, but the mission is now to build a real open-source NGFW for public benefit. The project must not depend on paywalled features or a future closed edition.

## Decision

Phragma is 100% open source.

- No open-core model.
- No BSL enterprise repository.
- No entitlement logic.
- No paywalled features.
- Fleet management, centralized logging, SSO, compliance reporting, and curated content tooling remain open source when built.
- Commercial work is limited to services around the open project.

## Consequences

The control-plane API is no longer a monetization seam. It is an ecosystem and stability seam.

Community trust improves, but the project must be disciplined about funding through support, services, certified builds, hosted optional services, and partnerships.
